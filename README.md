## Installation
Étapes à suivre pour installer le projet localement :

* Cloner le dépôt


```bash 
git clone https://github.com/nom_du_depot.git
```

* Installer les dépendances

```bash
npm install
```

* Lancer l'application

```bash
npm run dev
```

## Technologies utilisées

* React.js
* Vite.js
* Tailwind CSS
